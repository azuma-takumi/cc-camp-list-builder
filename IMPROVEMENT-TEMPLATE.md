# 改善ログテンプレ

案件ごとの `IMPROVEMENTS.md` や、横断メモに追加するときのテンプレです。

## 最小版

```md
## テーマ名

- 背景:
- 問題:
- 対応:
- 効果:
- 横展開:
- 状態:
```

## 案件別ログ向けテンプレ

```md
# 改善ログ

## 案件概要

- 案件名:
- 主な対象:
- 主な出力先:

## 改善一覧

| テーマ | 問題 | 対応 | 効果 | 横展開 |
|---|---|---|---|---|
| 例 | 例 | 例 | 例 | 例 |

## 詳細メモ

### 1. テーマ名

- 背景:
- 対応:
- 学び:

## 今後この案件で標準化したいこと

- 
```

## 横断まとめ向けテンプレ

```md
# 横断の改善・学び一覧

対象案件:

- 案件A
- 案件B

## 一覧

| テーマ | 学び | どちらの案件で出たか | 効果 | 横展開先 | 状態 |
|---|---|---|---|---|---|
| 例 | 例 | 例 | 例 | 例 | 採用 |

## テーマ別メモ

### テーマ名

- 代表例:
- 学び:

## 今後の標準ルール案

- 
```

## 使い分けの目安

- 案件固有の経緯や改善履歴は `IMPROVEMENTS.md`
- 複数案件で再利用できる学びは `CROSS-PROJECT-LEARNINGS.md`
- まだ検証中の内容は `状態: 保留`
- 再利用価値が低い単発修正は、一覧には入れず案件別ログだけに残す
